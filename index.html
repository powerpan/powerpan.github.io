好的代码，
是写给机器的诗

训练一个神经网络，就像人类学习的过程——从无知到感知，从犯错到成长。每一次反向传播，都是一次自我认知的迭代。

PyTorch Neural Network Backpropagation
train.py
import torch, torch.nn as nn, torch.optim as optim
# 1. 初始化权重 — 我们生来就带着某些天赋（随机初始化）
model = nn.Sequential(
nn.Linear(784, 256), nn.BatchNorm1d(256), nn.ReLU(),
nn.Linear(256, 128), nn.BatchNorm1d(128), nn.ReLU(),
nn.Linear(128, 10) # 输出 10 个类别的概率分布
)
optimizer = optim.AdamW(model.parameters(), lr=1e-3, weight_decay=1e-4)
criterion = nn.CrossEntropyLoss(label_smoothing=0.1)
for epoch in range(100):
# 2. 前向传播 — 用现有知识理解世界
predictions = model(train_images) # shape: [batch, 10]
# 3. 计算损失 — 认识到自己的不足
loss = criterion(predictions, train_labels) # 与真实标签对比
# 4. 反向传播 — 从错误中学习，计算每个权重的梯度
loss.backward() # dL/dW 自动求导
# 5. 更新权重 — 成长和改变，沿梯度方向微调参数
optimizer.step() # W = W - lr * grad
optimizer.zero_grad() # 清空梯度，准备下一轮
# 最终，我们不是要完美复制过去，而是要学会泛化到未来
