Overview

边缘 AI 的极致性能

PixelMind SDK 是一个为资源受限的边缘设备设计的计算机视觉推理库。核心引擎仅 5MB，却能在 Jetson Nano 上跑出 100+ FPS 的实时推理速度。

SDK 采用插件化架构，支持 ONNX、TensorRT、OpenVINO 等多种推理后端。内置的自动量化工具链可以将 FP32 模型一键转换为 INT8，精度损失 <1%。

Code Example

C++ SDK 用法

main.cpp
#include <pixelmind/engine.h>
int main() {
pm::Engine engine;
engine.load_model("yolov8n.onnx");
engine.set_backend(pm::Backend::TensorRT);
cv::Mat frame = cv::imread("input.jpg");
auto results = engine.infer(frame);
for (const auto& det : results.detections) {
printf("%s: %.2f\n", det.label, det.confidence);
}
return 0;
}
