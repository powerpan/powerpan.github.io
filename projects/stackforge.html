StackForge CLI

全栈项目脚手架工具，一条命令生成 Vue 前端 + 后端 API + Docker 配置，让团队从第一天就站在最佳实践之上，告别重复搭建项目结构的烦恼。

概述

为什么需要 StackForge？

每个新项目都要重新搭建脚手架——选择框架、配置 ESLint、设置 CI/CD、编写 Dockerfile、配置环境变量……这些重复劳动消耗了大量宝贵时间。

StackForge CLI 将这些最佳实践封装成可组合的模板，一条命令即可生成生产级项目骨架。支持 Vue + PHP/Python 技术栈，自动生成前端项目结构、后端 API 模板、Docker 配置和 CI/CD 流水线。

内置 20+ 种项目模板，覆盖 Vue + Flask、Vue + Spring Boot、Vue + PHP Laravel 等主流开发场景。核心引擎用 Python 编写，模板系统基于 YAML 配置，保持灵活性和可扩展性。

使用示例

快速上手

只需一条命令，即可初始化一个 Vue + PHP 全栈项目。StackForge 会交互式引导你选择技术栈，并自动生成所有配置文件。

terminal
$ pip install stackforge
$ stackforge init my-app --template vue-laravel
⚡ StackForge v1.2.0
✓ Scaffolding Vue 3 project structure...
✓ Generating Laravel API backend...
✓ Creating Docker configuration (dev + prod)
✓ Setting up GitHub Actions CI/CD
🎉 Project ready!
cd my-app && docker-compose up -d
stackforge.config.yaml
name: my-app
template: vue-laravel
frontend:
framework: vue3
typescript: true
pinia: true
backend:
framework: laravel
database: mysql
auth: jwt
devops:
docker: true
ci: github-actions
